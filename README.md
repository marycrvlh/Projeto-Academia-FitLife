# FitLife Academia 

Projeto desenvolvido durante as aulas do Curso Técnico de Desenvolvimento de Sistemas com o objetivo de praticar conceitos de HTML e CSS na criação de uma landing page para uma academia fictícia chamada **FitLife Academia**.

O site apresenta informações sobre a academia, planos disponíveis, estrutura do espaço e um formulário de contato moderno.

## 📋 Funcionalidades

* Página inicial com seção hero
* Menu de navegação
* Seção de benefícios da academia
* Cards com planos de assinatura
* Formulário de contato
* Layout moderno

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3

## 🎯 Objetivo do Projeto

Este projeto foi criado para desenvolver habilidades em:

* Estruturação de páginas com HTML
* Estilização com CSS
* Organização de layout
* Desenvolvimento de interfaces modernas
